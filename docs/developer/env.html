6.2 Term2里lrzsz支持文件上传

brew install lrzsz

创建2个bash脚本

/usr/local/bin/term_send.sh
#!/bin/bash
# Author: Matt Mastracci (matthew@mastracci.com)
# AppleScript from http://stackoverflow.com/questions/4309087/cancel-button-on-osascript-in-a-bash-script
# licensed under cc-wiki with attribution required
# Remainder of script public domain

FILE=`osascript -e 'tell application "iTerm" to activate' -e 'tell application "iTerm" to set thefile to choose file with prompt "Choose a file to send"' -e "do shell script (\"echo \"&(quoted form of POSIX path of thefile as Unicode text)&\"\")"`
if [[ -z "$FILE" ]]; then
    echo Cancelled.
    # Send ZModem cancel
    echo -e \\x18\\x18\\x18\\x18\\x18
    sleep 1
        echo
    echo \# Cancelled transfer
else
    echo $FILE
    /usr/local/bin/sz "$FILE" -E -e -b
    sleep 1
    echo \# Received $FILE
fi
/usr/local/bin/term_recv.sh
#!/usr/bin/env bash
# vim: fdm=marker foldlevel=0 sw=2 ts=2 sts=2

FILE=`osascript -e 'tell application "iTerm" to activate' -e 'tell application "iTerm" to set thefile to choose folder with prompt "Choose a folder to place received files in"' -e "do shell script (\"echo \"&(quoted form of POSIX path of thefile as Unicode text)&\"\")"`

if [[ -z "$FILE" ]]; then
  echo " Cancelled"
  # Send ZModem cancel
  echo -e \\x18\\x18\\x18\\x18\\x18
  sleep 1
  echo
  echo " # Cancelled transfer"
else
  cd "$FILE"
  "${HOMEBREW_PREFIX:-/usr/local}/bin/rz" -E -e -b
  sleep 1
  echo
  echo " # Sent -> $FILE"
  echo
fi

设置Iterm2的Tirgger特性，profiles->default->editProfiles->Advanced中的Tirgger

添加两条trigger，分别设置 Regular expression，Action，Parameters，Instant如下：

第一个trigger
Regular expression: rz waiting to receive.\*\*B0100
Action: Run Silent Coprocess
Parameters: /usr/local/bin/term_send.sh
Instant: checked
第二个
Regular expression: \*\*B00000000000000
Action: Run Silent Coprocess
Parameters: /usr/local/bin/term_recv.sh
Instant: checked

6.4 U盘和有些移动硬盘

一般我们的u盘的文件格式是ntfs,而mac不支持

# 插入u盘或移动硬盘后
diskutil list
# 看到如下的盘符
# Microsoft Basic Data 之后的就是你的盘符的名称了
# 有多少个你都记下来( 一般我们会将移动硬盘分区,就是分区的名称了)
/dev/disk2 (external, physical):
Microsoft Basic Data XYZ       1.0 TB     disk2s1

修改/etc/fstab, 添加如下 = 号右边的第一个就是写你的盘符名称,有多少个,就有多少行

LABEL=OPQ none ntfs rw,auto,nobrowse
LABEL=XYZ none ntfs rw,auto,nobrowse

映射盘符,将盘符映射到桌面快捷键方式一样

sudo ln -s /Volumes/XYZ ~XYZsktop/XYZ

重启电脑

6.4.1 遇到复制到u盘文件变灰色

# 表示删除文件或者文件夹里面的文件的 属性 com.apple.FinderInfo
# -d 表示删除, -r递归
xattr -d -r com.apple.FinderInfo *

6.4.2 聚焦搜索

# 打开
sudo mdutil -a -i on
# 关闭, 这样会搜不到软件
sudo mdutil -a -i off
