2 vimdiff时的快捷键

快捷键 功能
ctrl + w ,再按 l 切换到右侧 切换到右边的窗口
ctrl + w ,再按 h 切换到左侧 切换到左边的窗口

3 ipad esc 键怎么按

  • 没有 这个键， 用 ctrl+[ 或 cmd+.来代替 ESC键
  • 或者在 设置中 通用->键盘->实体键盘->修饰键 改大写锁定键=>ESC

4 nerdtree

  • 窗口切换快捷键
ctrl + w ,再按 h 切换到左侧
ctrl + w ,再按 l 切换到右侧
ctrl + w ,再按 w 自动切换到另外一边
