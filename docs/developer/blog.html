3.2.2 部署

3.2.3 国内无法访问的问题

需要配置域名

  1. 打开 域名解析页面(这里以aliyun为例)

  2. 点击 你想要使用的域名 那行的{==解析设置==},添加记录

  3. 输入

    选项 输入或选择的内容
    记录类型 CNAME
    记录类型 随便, 作为你部署的站点的域名
    记录值 cname.vercel-dns.com

  4. 在该dashboard页面,点击你部署的项目

  5. 选择settings->Domains->输入你的域名–>点击ADD

  6. 等待vercel验证你的域名,然后就可以在国内网络上访问了.
